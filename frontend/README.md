# ✈️ SkyLooms - Frontend

The React-based frontend for the SkyLooms premium airline booking system.

## 🚀 Quick Start

1. Install dependencies:
   ```bash
   npm install
   ```
2. Start development server:
   ```bash
   npm run dev
   ```

## 🏗️ Technical Details

- **Framework**: React 19 + Vite
- **Styling**: Vanilla CSS (Premium Glassmorphism)
- **State**: Global Context API (BookingContext, etc.)
- **Routing**: React Router 7

For full project deployment and backend details, see the [main README](../README.md).
